Para iniciar un proyecto GIT necesitamos ir, a travez de la consola, a la carpeta del proyecto.
Luego ejecutar:

git init

Luego de hacer cambios y/o crear nuevos archivos podemos ver el estado consola

git status

Para enviar los nuevos archivos al stagging

git add <nombre_archivo,...>

o

git add .

Para agregar todos los archivos nuevos


Para guardar los cambios en el repositorio se usa (Esto seria una version, un punto a donde podemos ir en cualquier momento)

git commit -m "<Comentario>"

Para agregar al stagging y guardar cambios en el repositorio

git commit -am "<Comentario>"


Para configurar el user y email

git config --global user.name "<Nombre>"
git config --global user.email "<Email>"


Para ver la configuracion actual

git config --list


Para agregar nuestro proyecto a un repositorio remoto

git remote add origin <url>

para ver el repositorio remoto

git remote --v

Para enviar los cambios al repositorio remote

git push -u origin <rama>
